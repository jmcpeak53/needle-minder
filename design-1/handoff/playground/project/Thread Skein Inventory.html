Thread Skein Inventory — wireframes

Low-fi sketches for an Android app that lets serious crafters photograph skein labels, capture brand + color code, browse & filter a stash, quick-deduct as threads get used, and export to CSV. Three variations on each key screen — tap a tab to switch.

Where users land on open. Two home variants (at-a-glance dashboard vs. jump-straight-to-inventory), plus a sitemap showing how the flows connect.
